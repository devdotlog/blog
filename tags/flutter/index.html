Flutter에서 모든건 Widget이다. Show/Hide까지도.

2020-09-11
flutter에서 모든 건 Widget입니다. 간단한 문자를 출력하는 Text부터, 다른 Widget을 담기 위한 Container까지도 모두 Widget입니다. 한마디로 사용자에게 보이든, 보이지 않든 화면을 구성하는 모든 것은 Widget입니다. 그리고 한발 더 나아가서 Widget 자체의 Show, Hide 까지도 별도의 Widget(Visibility)입니다. 다른 UI에서 Visibility UI를 다루는 다른 프레임워크를 보면 일반적으로 A라는 컴포넌트를 사용자에게 Show, Hide 처리하는 것은 그 컴포넌트 자체에서 show(), hide() 함수(혹은 메소드)를 제공하거나 투명도를 0으로 설정하여 처리하곤합니다. 예를 들어 안드로이드에서 TextView는 xml에서 visibility를 설정할 수 있습니다.

Web App을 Release 모드로 build Web Server 실행 테스트 Release 모드로 build 대상 프로젝트로 이동하고 command 라인에서 다음 명령을 실행하여 빌드합니다. $ flutter build web 정상적으로 빌드가 되었다면 프로젝트 폴더 내에 프로젝트폴더/build/web 디렉토리에 index.html 파일을 포함한 release 파일들이 생성된 것을 확인할 수 있습니다.
